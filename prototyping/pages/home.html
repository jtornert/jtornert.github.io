Demo Prototype

This site showcases the prototyping capabilities with htmx and Alpine.js.

Benefits

  • With htmx, you can quickly simulate responses from web servers and hypermedia APIs.
  • With Alpine.js, you can easily add trivial UI behaviour contained in both the initially loaded site as well as the API hypermedia loaded with htmx.
